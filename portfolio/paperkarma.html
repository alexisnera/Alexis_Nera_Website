Bates Paper Karma App

As part of a regional initiative to reduce the amount of paper consumption, Bates built a program that discourages the use of paper in the office. As a tool for monitoring this program and also as a tool for “positive peer pressure,” a website and web app is developed. This is ideal since the program encompasses use in all offices throughout the region. Each office is assigned specific tasks in building this program. The task of designing the web app is given to me. This web app is designed with heavy attention on utility. The design goal is to communicate with a universal language given that different offices in Asia will be the end user. The interface must also carry the new image of the company.
